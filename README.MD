## 🌟 GitHub 新手指南

### 第一步：创建仓库

1.  点击右上角 ➕ 选择 "New repository"
2.  输入仓库名 (建议小写+连字符)
3.  勾选初始化 README
4.  选择开源协议 (推荐 MIT)

### 第二步：协作开发

    # 克隆仓库
    git clone https://github.com/username/repo.git

    # 创建特性分支
    git checkout -b my-feature

    # 提交更改
    git commit -am "add awesome feature"

    # 推送分支
    git push origin my-feature

### 第三步：发起 Pull Request

1.  进入 GitHub 仓库页面
2.  点击 "Compare & pull request"
3.  填写 PR 描述模板
4.  关联相关 Issues
5.  等待代码审查 ✅
